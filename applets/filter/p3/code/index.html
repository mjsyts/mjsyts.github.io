An allpass filter passes all frequencies at equal amplitude — the frequency response is always 0 dB. Its effect is on phase only. The cutoff frequency is where the phase shift is exactly 180°. Q controls how steep the phase transition is around that point. Allpass filters are used in phasers, comb filters, and speaker time-alignment.
